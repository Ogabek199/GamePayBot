GamePayBot — Telegram WebApp (scaffold)

This repository contains frontend (Next.js) and backend (NestJS) scaffolding for the GamePayBot Telegram WebApp.

Start here:
- backend/: NestJS API
- prisma/: Prisma schema
- apps/web/: Next.js (mobile-first) WebApp
- apps/admin/: Next.js Admin Panel

.env.example contains required environment variables.

Next steps:
1. Install dependencies in backend and apps/web
2. Generate Prisma client
3. Run backend and frontend locally
